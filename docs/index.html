Projects

Pomodoro Timer (JS Study)

Early on in my web-development studies, I learned of the Pomodoro Technique. I began using it to maximize efficiency as I furthered my studies. This app is a helpful tool for completing a day's "to-do" list with a built-in timer for breaking up tasks into smaller sections.

Git Command Reference (JS Study)

I used this project to start using JavaScript for a functional project. I created a reference tool for Git commands so that I could look them up when I needed them. The commands are saved in Local Storage, so it's not an incredibly useful system unless I'm using the same browser and computer, but the project works as intended!

jQuery Study

I used this project to get my feet wet with jQuery. I practiced employing the most common jQuery methods and also gained more experience with responsive design.

CSS Study

You're looking at it! My goal was to practice using CSS Grid formatting to create a sleek landing page for my portfolio. I also explored setting my branding colors as CSS root variables and experimented with different ways to display photos.

CSS Flexbox vs Grid Study

I wanted to gain a deeper understanding of how Flexbox and Grid work in CSS. I created a Pinterest-like "Apartment Inspo Board" to practice. I also worked on making this site responsive to different sized viewports to work on formatting with both layouts.

HTML Study

This study was created using (almost exclusively) HTML. It's not the prettiest project, but it gives examples of the most widely-used HTML elements.
